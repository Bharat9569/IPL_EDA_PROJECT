# IPL EDA Project

## Overview

This project is part of my AI Engineer Journey (Week 3). The goal of this project is to perform Exploratory Data Analysis (EDA) on the IPL dataset using Python and understand how to extract insights from real-world data.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Google Colab

## Analysis Performed

* Team-wise match participation
* Team win percentage
* Toss winner vs match winner analysis
* Toss decision trends (Bat vs Field)
* Venue analysis
* Season-wise match trends
* Data visualization using Matplotlib and Seaborn

## Key Learnings

* Data cleaning and preprocessing
* Working with Pandas DataFrames
* Data aggregation using groupby and value_counts
* Creating visualizations with Matplotlib and Seaborn
* Extracting meaningful insights from data
